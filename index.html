Commonslens: Wikimedia Discovery Engine

An advanced visual discovery engine for researchers, designers, and developers to explore Wikimedia Commons.

Features

  • Semantic Search: Find media depicting specific subjects (e.g., Q146 for Cat).
  • License Filtering: Filter by CC0, CC BY-SA 4.0, and Public Domain rights.
  • Geolocation: Search for media coordinates using Latitude and Longitude.
  • Quality Assessment: Discover Featured Pictures, Quality Images, and Valued Images.

About

Powered by the CirrusSearch API and Wikidata, this tool provides a high-performance visual interface to execute complex metadata queries without writing SQL.
